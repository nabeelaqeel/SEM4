### Configure SSH on R1

```
ip domain-name r1.com

crypto key generate rsa general-keys modulus 1024

username user01 secret user01pass

line vty 0 4
login local 
transport input ssh
exec-timeout 5 0

login block-for 300 attempts 4 within 120
```
